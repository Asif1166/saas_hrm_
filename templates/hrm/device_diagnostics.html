ZKTeco Troubleshooting Guide
Network Issues
  • Ensure device is powered on and connected to network
  • Check IP address configuration on device
  • Verify device and server are on same network
  • Check firewall settings (port 4370 should be open)
  • Try pinging the device from server
Protocol Issues
  • Check device TCP/IP settings in device menu
  • Ensure device supports TCP/IP communication
  • Try different device ID (1, 2, 3, etc.)
  • Update device firmware if possible
  • Check device communication protocol settings
Device Configuration Steps
  1. Access device menu (usually by pressing Menu or * key)
  2. Go to Communication or Network
  3. Set IP Address to static IP (e.g., 192.168.68.3)
  4. Set Subnet Mask (usually 255.255.255.0)
  5. Set Gateway (your router IP)
  6. Set Port to 4370
  7. Save settings and restart device
